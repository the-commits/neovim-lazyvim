A customized Neovim setup based on [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.

## Features

- **Opencode Integration**: AI-powered coding assistant with dedicated keymaps
- **Debugging Support**: Integrated debugging with DAP (Debug Adapter Protocol)
- **Smart Formatting**: Automatic code formatting with language-specific tools
- **Linting**: Integrated linting for multiple languages
- **Git Integration**: Seamless Git workflow with LazyGit integration

## Key Mappings

### Opencode
- `<leader>og` - Toggle Opencode UI
- `<leader>oi` - Open input window
- `<leader>oo` - Open output window
- `<leader>od` - Open diff view

### Debugging
- `<leader>b` - Toggle breakpoint
- `<leader>dr` - Run/Continue debugging
- `<leader>ds/di/do` - Step over/into/out
- `<leader>du` - Toggle debug UI

### Git
- `<leader>mg` - LazyGit terminal
- `<leader>gdiff` - Toggle git diff split

### Project Navigation
- `<leader>fp` - Find project files
- `<leader>fw` - Find words in project
